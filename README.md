# 📚 Lista de Exercícios HTML & CSS (91 Atividades)

> Coleção de 91 exercícios práticos desenvolvidos para aprimorar habilidades em HTML, CSS e fundamentos de desenvolvimento web.

---

## 📌 Sobre o Projeto

Este repositório reúne uma lista de **91 exercícios de HTML e CSS**, propostos durante o programa **PROA**, com o objetivo de reforçar a prática e consolidar conhecimentos essenciais no desenvolvimento front-end.

As atividades abrangem desde conceitos básicos até construções mais estruturadas de layout, permitindo evolução progressiva ao longo dos exercícios.

---

## 🎓 Contexto Acadêmico

Projeto desenvolvido no programa **PROA**, com foco na formação prática em tecnologia e desenvolvimento web.

A lista foi proposta pelo professor como forma de:

* Fixar fundamentos de HTML e CSS
* Estimular a prática contínua
* Desenvolver organização de código
* Evoluir na construção de interfaces
